# NovaTech Login

Este projeto foi desenvolvido com o objetivo de praticar e consolidar conhecimentos sobre autenticação básica em aplicações web. A ideia foi construir um sistema simples de login, conectando uma interface em HTML, CSS e JavaScript a um servidor Node.js responsável por validar e responder às requisições.

## Objetivo do Projeto

O foco principal foi entender na prática como funciona o fluxo de autenticação entre cliente e servidor. Ao longo do desenvolvimento, exercitei:

- Estruturação de páginas e formulários de login  
- Validação de dados no frontend  
- Configuração de um servidor em Node.js  
- Criação de rotas para receber e processar requisições  
- Retorno de respostas ao cliente com base na validação  
- Organização de arquivos e lógica entre frontend e backend  

## Tecnologias Utilizadas

- HTML  
- CSS  
- JavaScript  
- Node.js  

## Aprendizados

Este projeto me ajudou a compreender melhor o fluxo completo de autenticação, desde a captura dos dados no formulário até o processamento no backend. Também foi uma ótima oportunidade para treinar organização de código, boas práticas e integração entre camadas da aplicação.
